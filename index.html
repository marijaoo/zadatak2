Web development

Na Web Design programu naučićete načine izrade i postavljanja web sajtova. polaznici će se upoznati sa osnovama HTML jezika, CSS stilovima, JavaScriptom, ali i najmodernijim tehnikama HTML5 animacije i prikaza sadržaja, uz optimizaciju responsive dizajna za računare, tablete, telefone.
